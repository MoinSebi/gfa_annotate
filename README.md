# gfa_annotate
Overlay gene annotation from a gff format to the genome graph. 

## Output 
| Col | Type         | Description |
|-----|--------------|-------------|
| 1   | int          | Node id     |
| 2   | String list  | Type        |
| 3   | String list  | Genes       |
